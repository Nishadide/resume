Illegal Parking Detection using Computer Vision

Final Year Research Project The aspect of this research is to give a solution to mitigate parking violations by using machine learning and image processing in Sri Lanka. We having four group memaber and everyone plane to build machine learning models under four categories. I'm responsible for building a machine learning model to detect parking violations considering road signs and road lines. Apart from that detect illegal parking by considering traffic lights, by roads, vehicles more towards to road, parked infront of gates are plan to be done. The novalty of this project is detecting illegal parking not in specified or marked area but in any place it can be occurs and predict free parking areas.
